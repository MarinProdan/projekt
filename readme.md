Klub FBC liberec

**Žák**
Marin Prodan

**Stručný popis tématu**
Webová prezentace bude zaměřena na florbalový klub FBC Liberec. Stránky budou obsahovat informace o historii klubu, zajímavostech a cenách. Součástí budou také aktuální termíny tréninků pro nové zájemce. Cílem je vytvořit přehledný web, který poskytne základní informace pro hráče i fanoušky.
